Вариант 1: Онлайн на DB Fiddle (рекомендуется для изучения)
Самый простой способ начать практику без установки PostgreSQL:

📖 Откройте DB Fiddle
⚙️ Выберите PostgreSQL в качестве базы данных
📋 Скопируйте содержимое 01_ddl_schema.sql в левую панель (Schema SQL)
📋 Скопируйте содержимое 02_mock_data.sql в ту же левую панель (после схемы)
🔍 Пишите SQL-код в правой панели (Query SQL)
▶️ Нажмите Run для выполнения
Преимущества DB Fiddle:

✅ Не требует установки PostgreSQL
✅ Мгновенный старт
✅ Можно делиться ссылками на запросы
✅ Идеально для обучения и практики

🗃️ Схема базы данных
Основные таблицы
products - Каталог товаров с ценами и себестоимостью
campaigns - Маркетинговые кампании с затратами, показами и кликами
users - Пользователи, связанные с кампанией привлечения (NULL = органика)
user_sessions - Ежедневная активность пользователей для метрик вовлеченности
orders - Транзакции покупок
order_items - Детализированные позиции заказов
📊 Реализованные метрики
1. CPM (Стоимость за тысячу показов)
Формула: (Общие рекламные затраты / Общие показы) × 1000

Бизнес-влияние: Измеряет эффективность затрат на кампании по повышению узнаваемости бренда.

2. CPC (Стоимость за клик)
Формула: Общие рекламные затраты / Общие клики

Бизнес-влияние: Измеряет эффективность затрат на привлечение трафика. Критически важно для оптимизации performance-маркетинга.

3. CTR (Коэффициент кликабельности)
Формула: (Общие клики / Общие показы)

Бизнес-влияние: Измеряет релевантность рекламы и эффективность креативов. Высокий CTR указывает на лучшее таргетирование и сообщения.

4. CAC (Стоимость привлечения клиента)
Формула: Затраты на кампанию / Количество привлечённых пользователей

Бизнес-влияние: Критически важно для понимания юнит-экономики. Должно быть меньше LTV для прибыльного роста.

Модель атрибуции: Прямая связь пользователя с кампанией через users.campaign_id. NULL означает органический трафик.

5. ROMI (Возврат на маркетинговые инвестиции)
Формула: ((Валовая прибыль - Маркетинговые затраты) / Маркетинговые затраты) Где Валовая прибыль = Выручка - Себестоимость товаров

Бизнес-влияние: Измеряет прибыльность маркетинговых кампаний. Положительный ROMI указывает на прибыльные маркетинговые затраты.

6. DAU/MAU (Коэффициент липкости)
Формула: Среднее DAU за месяц / MAU за тот же месяц

Бизнес-влияние: Измеряет липкость продукта (sticky factor). Как часто пользователи возвращаются к продукту.

7. Retention Rate (Коэффициент удержания)
Формула: Пользователи, вернувшиеся через N дней / Общее количество новых пользователей

Бизнес-влияние: Измеряет способность продукта удерживать пользователей. Ключевые периоды: Day 1, Day 7, Day 30 retention.

8. ARPU (Средняя выручка с пользователя) и ARPPU (Средняя выручка с платящего пользователя)
Формула ARPU: Общая выручка / Общие пользователи Формула ARPPU: Общая выручка / Только платящие пользователи

Бизнес-влияние: Ключевые метрики для эффективности монетизации и потенциала роста выручки.

9. LTV_30d (30-дневная жизненная ценность)
Формула: Общая выручка в первые 30 дней / Количество пользователей в когорте

Бизнес-влияние: Ранний индикатор ценности клиента. Используется для оптимизации затрат на привлечение и прогнозирования долгосрочной ценности.

💡 Практические задания
📊 Задания для анализа каналов привлечения
Вопросы для исследования:

Какой канал самый эффективный по CAC? Рассчитайте стоимость привлечения клиента для каждого канала
Какой канал приносит наибольшую LTV? Сравните 30-дневную ценность пользователей по каналам
Где лучший ROMI? Проанализируйте возврат инвестиций по типам кампаний
Качество vs Количество: Сопоставьте объем трафика и конверсию в покупки
📈 Задания для анализа метрик вовлеченности
Вопросы для исследования:

Паттерны активности: Как меняется DAU/MAU/Sticky factor по месяцам? Есть ли сезонность?
Сегментация по устройствам: Отличается ли поведение пользователей мобильных и десктопных устройств?
💰 Задания для анализа монетизации
Вопросы для исследования:

ARPU vs ARPPU: Как меняются метрики по месяцам? Какая доля пользователей совершает покупки?
Корзина покупок: Какие самые популярные категории товаров, а какие самые прибыльные? Какой средний чек корзины?
Временные паттерны: В какие дни недели/время суток больше заказов?
Повторные покупки: Какой процент пользователей покупает повторно в течение 30 дней?
